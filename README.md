# Country Explorer Dashboard

## Setup
1. Clone repo
2. npm install
3. Create .env file
VITE_WEATHER_API_KEY=
4. npm run dev

## Features
- Country listing with pagination
- Search & filters
- Country details page
- Weather integration
- Favorites (LocalStorage)
- Responsive UI

## Tech Stack
React, TypeScript, Axios, React Router, Context API
